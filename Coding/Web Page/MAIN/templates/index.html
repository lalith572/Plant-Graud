An Integrated System
to make Revolution in Agriculture

Our system uses IoT technologies to monitor environmental conditions and automates irrigation systems based on real-time data from the sensors. Data are collected and received by NodeMCU which can be linked to an interactive app that shows the real-time values along with the standard values of different factors required by a crop. Additionally, a machine learning-based model that helps in classifying and detecting diseases or infections using a trained model. The module leverages machine learning method to recognize various plant diseases, enabling early intervention and minimizing crop damage, and also provides preventive measures for the particular disease. The main targets of our system are terrace farming and small-scale farming. Integrating both the IoT system and machine learning module will provide a comprehensive solution for precision agriculture, enhancing crop productivity while conserving resources. Thus, our integrated system will provide the best results in the farming world.

Some common questions often asked

For what specific plant does the system provide solution?

Our System currently detect diseases and provides solutions for Tomato plant and in future we will provide for more plantsSo Stay Connected.

What are the Sensors used in this project?

Our system has a Soil Mositure sensor,DHT11 Sensor,PIR Motion Sensor,Water Flow Sensor.

Which MicroController is used in this Project?

In this project,we have used NodeMCU ESP8266 instead of Arduino or Raspberry PI since it is cost effective and has in-built Wi-Fi module.

What are the Algorithms used for training the model?

We have try different Algorithms like Transfer Learning,VGG16,Inception V3,CNN,Mobile Net V2 and in these algorithms CNN provides best acccuracy compared to others so it is used for disease detection .

Will the system provides cure or Preventive measures?

Yes,Our System will provide the preventive measures for the each particular diseases.

How much does the System Hardware Costs?

Our System Hardware costs around Rs: 1500 which depends on sensors cost at the time of purchase.
